# Epic: 结算引擎 (Settlement Engine)

- **ID:** E-004
- **状态:** 未开始
- **优先级:** P0
- **目标:** 自动算出"谁该给谁多少钱"，并优化转账路径

## 验收标准

- [ ] 自动计算每个成员的净收支（付了 - 应分摊）
- [ ] 输出最优转账路径（最少笔数）
- [ ] 生成结算单：总金额、人均、每人净收支、转账列表
- [ ] 成员可标记"已转账给 XX"
- [ ] 标记实时同步给所有成员
- [ ] 100 笔账目结算 < 1 秒
- [ ] 边界处理：所有人结清、某人结清、零金额不生成转账
- [ ] 🆕 支持**按组维度结算**（"张家 +800"、"李家 -500"）
- [ ] 🆕 结算单顶部可切换"按组查看"/"按人查看"
- [ ] 🆕 组内可展开查看个人明细
- [ ] 🆕 转账列表支持按组/按人两种粒度

## 备注

- **算法核心**: 贪心算法，最大债务人配最大债权人，min(|debt|, credit)
- **复杂度**: O(n log n)
- **可视化**: 转账关系图（V1.1 优化）
- **数据存储**: 每次结算生成快照，便于历史查询
- **隐私**: 转账记录仅旅程内成员可见
