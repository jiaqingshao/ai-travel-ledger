# Epic: 快速记账 (Expense Recording)

- **ID:** E-002
- **状态:** 未开始
- **优先级:** P0
- **目标:** 让用户 3 秒内完成一笔账的录入

## 验收标准

- [ ] 3 步内完成记账：付款人 → 类别 → 金额
- [ ] 默认付款人为上次付款人
- [ ] 默认类别为上次类别
- [ ] 支持备注（可选）
- [ ] 支持照片附件（可选，最多 3 张）
- [ ] 支持时间选择（默认当前，可改）
- [ ] 支持删除/编辑账目
- [ ] 账目列表按时间倒序展示
- [ ] 支持按类别筛选
- [ ] 离线可记账，恢复网络后自动同步
- [ ] 一只手操作友好

## 备注

- **3 秒原则**: 录入流程是关键 UX，必须打磨
- **数字键盘**: 不要用系统键盘，自定义数字键盘更快
- **照片压缩**: 上传前压缩到 < 500KB
- **离线优先**: 即使在没网络时也要能记，这是自驾场景核心
- **乐观更新**: UI 立即响应，后台异步同步
