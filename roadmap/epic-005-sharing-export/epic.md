# Epic: 分享导出 (Sharing & Export)

- **ID:** E-005
- **状态:** 未开始
- **优先级:** P1
- **目标:** 让用户能把结算单分享到微信，方便催账

## 验收标准

- [ ] 一键生成结算单图片（长图）
- [ ] 图片包含：旅程名、总金额、人均、转账列表
- [ ] 一键分享到微信 / QQ / 复制到剪贴板
- [ ] 支持导出 PDF
- [ ] 支持导出 CSV（详细账目）
- [ ] 图片美观、有品牌感
- [ ] 生成速度 < 3 秒

## 备注

- **品牌化**: 这部分关系到"看起来像不像大厂产品"
- **设计优先**: 建议先用 Figma 出模板，再写代码
- **微信限制**: 不能直接生成"小程序卡片"，分享图片是稳妥方案
- **隐私**: 分享前可让用户选择"匿名化"（隐藏昵称）
