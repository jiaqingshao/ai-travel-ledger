# Epic: 旅程管理 (Trip Management)

- **ID:** E-001
- **状态:** 未开始
- **优先级:** P0
- **目标:** 让用户能创建一次旅行的"账本空间"，并管理同行成员

## 验收标准

- [ ] 用户可创建新旅程（名称、起止日期、目的地、基础货币）
- [ ] 用户可添加/删除/编辑成员（昵称即可，注册可选）
- [ ] 用户可邀请未注册成员加入（分享链接 / 二维码）
- [ ] 用户可结束 / 归档旅程
- [ ] 旅程首页 Dashboard 显示：名称、日期、总花费、人均
- [ ] 多端同步：用户在一台设备创建，另一台设备可见

## 备注

- 邀请功能：MVP 可仅做"分享链接 + 手动输入昵称加入"，不做扫码
- 用户系统：MVP 不强制注册，昵称 + 设备 ID 即可
- 国际化：MVP 仅中文
- 性能要求：旅程列表加载 < 500ms
