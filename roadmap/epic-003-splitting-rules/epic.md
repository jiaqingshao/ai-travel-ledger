# Epic: 分摊规则 (基础) (Splitting Rules - Basic)

- **ID:** E-003
- **状态:** 未开始
- **优先级:** P0
- **目标:** 支持最常用的分摊方式，复杂规则留 V1.1

## 验收标准

- [ ] 支持**全员均摊**（默认）
- [ ] 支持**按比例分摊**（手动设置比例，总和=100%）
- [ ] 支持**部分人参与**（指定不参与的人）
- [ ] 实时显示每个成员应分摊金额
- [ ] 处理尾差（自动补偿给比例最大者）
- [ ] 分摊规则可在记账时选择
- [ ] 也可在账目详情页修改
- [ ] 默认推荐均摊（不强制选择）
- [ ] 🆕 支持**按组分摊**（指定参与的是哪些组，自动展开为成员）
- [ ] 🆕 支持**混合分摊**（部分组 + 部分个人）
- [ ] 🆕 按组分摊时，组内默认均摊

## 备注

- **MVP 范围**: 仅均摊 + 比例 + 部分人参与
- **复杂规则**（年龄段、按里程、固定金额）放到 V1.1 (E-007)
- **算法要点**: 尾差 = 总金额 - 每人金额 × 人数
- **测试覆盖**: 边界条件（0%、100%、奇数人分钱等）
