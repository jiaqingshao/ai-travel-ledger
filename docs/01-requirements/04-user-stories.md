# 用户故事 (User Stories)

> 用户故事格式：作为 [角色]，我希望 [功能]，以便 [价值]

---

## 一、旅程管理

### US-001: 创建旅程
**作为** 自驾游组织者，
**我希望** 快速创建一个新旅程并添加所有同行成员，
**以便** 我们有一个统一的记账空间。

**验收标准**:
- 3 分钟内可完成创建 + 添加 5 个成员
- 旅程名支持 50 字
- 成员数限制 15 人

### US-002: 邀请成员加入
**作为** 已加入旅程的成员，
**我希望** 邀请还没下载 App 的朋友加入，
**以便** 大家都能记账和查看结算。

**验收标准**:
- 支持分享链接 + 二维码
- 对方通过链接输入昵称即可加入
- 实时同步到所有成员

### US-003: 结束旅程
**作为** 组织者，
**我希望** 在旅行结束后归档旅程，
**以便** 我能专注于新旅程，历史记录仍可查看。

**验收标准**:
- 一键归档
- 归档后变为只读
- 历史列表可查询

## 二、快速记账

### US-010: 快速记录一笔账
**作为** 付款人，
**我希望** 在付完款后 30 秒内记录下这笔费用，
**以便** 不会遗忘，不会算错。

**验收标准**:
- 3 步内完成（付款人→类别→金额）
- 支持拍照附件
- 支持备注

### US-011: 查看账目列表
**作为** 旅程成员，
**我希望** 看到本次旅程的所有账目，
**以便** 了解花费情况。

**验收标准**:
- 按时间倒序展示
- 支持按类别筛选
- 显示总金额

### US-012: 编辑 / 删除账目
**作为** 付款人，
**我希望** 能修改或删除我记的账，
**以便** 修正错误。

**验收标准**:
- 只有付款人或组织者可改/删
- 删除需二次确认
- 不可修改 24h 前的账（防作弊，可配置）

---

## 三、分摊规则

### US-020: 设置均摊
**作为** 组织者，
**我希望** 一笔账默认全员均摊，
**以便** 大多数场景下不用操作。

**验收标准**:
- 默认均摊
- 显示每人应分摊金额

### US-021: 设置按比例分摊
**作为** 组织者，
**我希望** 油费按车辆/里程分摊，
**以便** 公平合理。

**验收标准**:
- 可设置每个人的比例
- 自动归一化（总比例=100%）

### US-022: 设置部分人参与
**作为** 组织者，
**我希望** 某些人没吃这顿饭就不分摊，
**以便** 精准结算。

**验收标准**:
- 默认全员参与
- 可勾选/取消参与者

## 四、结算引擎

### US-030: 查看结算单
**作为** 组织者，
**我希望** 一键生成结算单，
**以便** 知道谁该给谁多少钱。

**验收标准**:
- 包含每人净收支
- 包含最优转账路径
- 包含总金额、人均

### US-031: 标记已转账
**作为** 债务人，
**我希望** 转账后标记已还，
**以便** 不再被催。

**验收标准**:
- 一键标记
- 实时同步给所有人
- 历史可查

> **决策日期**: 2026-07-15

### US-013: 一句话记账
**作为** 付款人，
**我希望** 一句话说完账目信息（"刚才加油 200"），
**以便** 不用手动选类别、输金额。

**验收标准**:
- 一句话包含：付款人 + 类别 + 金额 + 可选备注
- 识别准确率 > 90%（中文普通话）
- 离线可用（系统 STT）
- 3 秒内完成识别 + 预览
- 识别后可手动修正

### US-014: 🆕 自动归类
**作为** 记账人，
**我希望** 语音记账时自动判断费用类别，
**以便** 不用手动选。

**验收标准**:
- 基于关键词 + 本地 LLM
- 覆盖 9 个常用类别
- 置信度 < 80% 时弹出选择

### US-015: 设置每周自动记账
**作为** 长期旅程组织者，
**我希望** 一次性设置"每周一记民宿费 800"，
**以便** 不用每周手动重复录。

**验收标准**:
- 周期：每天/每周/每月/每年
- 一次性设置，长期生效
- 可暂停/恢复
- 历史记录可查
- 提前 1 天推送通知（可选）

## 八、⏸️ 旅程统计 (E-010) — v0.3 制定，v0.3.1 暂缓至 V1.1 候选

> **状态**: ⏸️ 暂缓，详见 [ADR-004](../02-architecture/04-adr/ADR-004-prd-v0.3-p0-defer.md)
> **决策日期**: 2026-07-15

### US-016: 查看旅程统计图表
**作为** 旅程成员，
**我希望** 一屏看到本旅程的消费图表，
**以便** 了解钱花在哪了。

**验收标准**:
- 分类饼图（油费 30% / 餐饮 25% / 住宿 20% ...）
- 人均柱状图（谁花得多）
- 每日趋势折线
- 总金额 + 人均 + 最高/最低
- 一屏可看

### US-017: 旅程结束自动出统计报告
**作为** 组织者，
**我希望** 旅程归档时自动生成统计报告，
**以便** 发到群里让大家回顾。

**验收标准**:
- 一键生成统计图片
- 包含图表 + 文字总结
- 可分享到微信

## 九、离线与同步

### US-050: 离线记账
**作为** 自驾游用户，
**我希望** 在没有网络的地方也能记账，
**以便** 不会丢失账目。

**验收标准**:
- 完全离线可记账
- 网络恢复后自动同步
- 冲突时合理处理
