## 一、用户痛点

### 痛点 1: 算账太麻烦 ⭐⭐⭐⭐⭐
- **场景**: 自驾游结束后，群里发一堆"我付了 XX、你付了 YY"，算账半小时
- **典型台词**: "我出 1500、你出 800、AA 之后你给我 350..."
- **情绪**: 烦躁、不想算、想赶紧把钱结清

### 痛点 2: 算完发现算错了 ⭐⭐⭐⭐
- **场景**: 群里算账，要么有人漏算、要么小数点错位
- **典型台词**: "等等，再算一遍"
- **情绪**: 不信任、扯皮

### 痛点 3: 复杂场景算不清 ⭐⭐⭐⭐
- **场景**: 油费按里程分、住宿均摊、小孩半价、某人没参加某项活动...
- **典型台词**: "这个怎么算？"
- **情绪**: 无奈、放弃记账

### 痛点 4: 记录丢失 ⭐⭐⭐
- **场景**: 群里发的账，过了几天就翻不到了
- **典型台词**: "那个图谁还存着？"
- **情绪**: 后悔没记账

### 痛点 5: 旅途场景网络差 ⭐⭐⭐
- **场景**: 自驾途中到山区/高速无信号
- **典型台词**: "等有网了再记"
- **情绪**: 拖延，事后补不全

## 三、核心功能脑暴

### 🔴 P0 - MVP 必须有

1. **快速记账** — 3 秒内完成一笔
2. **多种分摊方式** — 至少均摊
3. **结算功能** — 自动算"谁给谁多少钱"
4. **旅程管理** — 把一次旅行的所有账目组织起来
5. **成员管理** — 添加/删除成员

### 🟡 P1 - V1.1 应该做

1. **分摊规则细化** — 比例分摊、按份数、部分人参与
2. **结算单分享** — 一键生成图片分享到微信
3. **历史记录** — 查看过去的旅程
4. **费用分类** — 油费/过路费/住宿/餐饮/门票...

### 🟢 P2 - 锦上添花

1. **OCR 识别小票**
2. **多币种**
3. **数据导入导出**
4. **企业版/报销**
5. **文件导入（其他软件生成的旅行计划）**

## 四、差异化方向

> 区别于市面上的产品，我们做什么？

| 维度 | 差异化 |
|---|---|
| **场景化** | 只做自驾游/团队游，不做泛记账 |
| **零摩擦** | 3 秒记账，不学就会 |
| **智能分摊** | 复杂场景有默认规则 |
| **结算优化** | 减少转账笔数 |
| **自驾专属** | 油费、过路费、停车费分类 |
| **离线可用** | 弱网/无网也能记 |

## 七、商业模式讨论

### 免费版 (MVP)
- 基础记账 + 均摊 + 结算
- 单次旅程最多 15 人
- 同时进行 3 个旅程

### 高级版 (V1.1+)
- ¥18/月 或 ¥168/年
- 解锁：复杂分摊、无限旅程、OCR、多币种

### 未来可能
- 企业版（按组织收费）
- 平台抽佣（接酒店/保险/租车）
