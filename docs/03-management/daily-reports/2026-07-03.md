## 🎨 Phase 4: UI 主色修正

**问题**：`ColorScheme.from(seedColor: 0xFF2E7DD7)` 跑出来还是浅绿色

**原因**：Material 3 `fromSeed` 算法在 Flutter 3.24 + Light 主题下生成了意外的浅绿色板

**修复**：不用 fromSeed，直接定义完整 ColorScheme：
```dart
colorScheme: const ColorScheme(
  primary: Color(0xFF3B4A6B),      // 深蓝灰 - AppBar
  secondary: Color(0xFFD4A574),     // 暖棕色 - 演示卡片
  surface: Color(0xFFF8F7F4),       // 米白色 - 背景
  ...
)
```

**效果（截图验证）**：
- AppBar: **深蓝灰色** (#3B4A6B 风格)
- 演示数据卡片: **暖棕色** (#D4A574 风格)
- FAB: **蓝灰色**
- 整体: **冷暖对比中性偏暖**
