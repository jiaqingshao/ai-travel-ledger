# 五子棋 (Gomoku)

一个使用 **HTML + CSS + JavaScript** 实现的网页版标准 15×15 五子棋游戏，纯前端、单页、无依赖。

## 📁 文件结构

```
gomoku/
├── index.html   # 页面骨架（棋盘 + 控制面板 + 落子记录）
├── style.css    # 样式（木纹棋盘 UI、棋盘面板、按钮）
├── script.js    # 游戏逻辑（绘制、落子、胜负判定、悔棋）
└── README.md    # 本文件
```

## 🚀 如何运行

### 方式一：直接打开

最简单：在文件管理器中双击 `index.html`，使用任意现代浏览器（Chrome / Edge / Firefox / Safari）打开即可开始游戏。

### 方式二：本地静态服务器（推荐）

部分浏览器在 `file://` 协议下会限制 Canvas，使用本地服务器体验更稳定：

```bash
# 进入目录
cd gomoku

# 任选其一
python -m http.server 8080
# 或
npx serve .
```

随后访问 http://localhost:8080

## 🎮 玩法

- **黑棋先手**，双方轮流落子
- 鼠标点击棋盘交叉点放置棋子（落点会有小点预览）
- **横 / 竖 / 主对角 / 副对角** 任一方向先连成 **五子** 即获胜
- 棋盘下满 225 子且无五连则判为平局

## 🕹️ 操作

| 操作 | 触发方式 |
|------|----------|
| 落子 | 鼠标点击交叉点 / 移动端触屏 |
| 悔棋 | 点击右侧「悔棋」按钮，或按键盘 `U` |
| 重新开始 | 点击右侧「重新开始」按钮，或按键盘 `R` |
| 预览落点 | 鼠标悬停时显示小点 |

## ✨ 功能特性

- ✅ 15×15 标准棋盘，含 5 个星位（天元 + 四角的 3/3/11/11）
- ✅ 木纹渐变棋盘 + 棋子径向渐变 + 阴影，立体观感
- ✅ 实时显示当前回合、步数、胜负信息
- ✅ 落子记录面板（编号、玩家、坐标 A1–O15）
- ✅ 最近一子红色标记
- ✅ 鼠标悬停落点预览
- ✅ 悔棋（支持游戏结束后继续悔到进行中）
- ✅ 响应式布局，移动端触屏可用
- ✅ 键盘快捷键（`U` 悔棋，`R` 重开）
- ✅ 胜负结算遮罩层

## 🧩 核心算法

胜负判定：对刚落下的子沿 4 个方向（横、竖、主对角、副对角）双向延伸计数，若连续同色棋子 ≥ 5 即判胜。复杂度 O(1)，单步即时判定。

```
score = 1
向前延伸 count++，直到越界或颜色不同
向后延伸 count++，直到越界或颜色不同
if (count >= 5) → 胜利
```
