# Epic: 语音记账 (Voice Recording)

- **ID:** E-008
- **状态:** 未开始
- **优先级:** P0（v0.3 新增）
- **目标:** 一句话完成记账，3 秒搞定，驾驶/单手场景友好

## 背景

2026-06-28 市场调研发现：
- 百事 AA 记账的 **AI 语音记账** 是国内同类最大创新点
- 来福记账 / 叨叨记账 / 有鱼记账 都在向"对话式记账"演进
- 用户痛点：开车/手忙/不熟键盘时无法快速记账

## 验收标准

- [ ] 一句话识别：付款人 + 类别 + 金额 + 可选备注
- [ ] 识别准确率 > 90%（中文普通话）
- [ ] 离线可用（Android 系统 STT `speech_to_text` 包）
- [ ] 自动归类（关键词 + 本地 LLM Qwen3.6 35B）
- [ ] 3 秒内完成识别 + 预览
- [ ] 识别结果预览 + 确认后才入库
- [ ] 支持方言识别（普通话优先，粤语/四川话 V1.1）
- [ ] 麦克风权限友好提示

## 技术选型

| 维度 | 方案 | 理由 |
|---|---|---|
| 语音识别 | Android `speech_to_text` (系统 STT) | 免费、离线、无需 API key |
| 自然语言理解 | 本地 LLM（Qwen3.6 35B）+ 关键词兜底 | 离线、隐私、低成本 |
| LLM 部署 | 通过 OpenClaw API (http://192.168.1.60:8033/v1) | 已有基础设施 |

## 备注

- **隐私优先**：语音不上传服务器，全部本地处理
- **降级策略**：识别失败 / 置信度低 → 提示手动
- **驾驶安全**：建议车机/车载场景使用，UI 大按钮、避免分心
- **方言扩展**：V1.1 再加粤语/四川话
